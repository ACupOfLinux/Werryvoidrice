# Werryvoidrice




# HOW TO INSTALL 
git clone https://github.com/ACupOfLinux/Werryvoidrice 

cd Werryvoidrice 

mv .config/* ~/.config 

# As simple as that :) 


# IMPORTANT 
I am using a custom picom fork from jonaburg that requires ninja and meson 
https://github.com/jonaburg/picom < This is the one you need to compile>
